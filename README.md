# ColdScore — AI Cold Email Grader

Free, instant cold email feedback powered by AI. No signup required.

**[Try it live →](https://michael-ami-01.github.io/coldscore/)**

## Features
- Paste your cold email, get an instant grade (A+ to F)
- Section-by-section breakdown: subject line, opener, value prop, CTA
- 3 specific, actionable improvements
- Works with your own API key (BYOK) or our free proxy
